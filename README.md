
# Workforce Attrition Patterns and Risk Hotspot Analysis at Palo Alto Networks

## Project Overview

This project analyzes employee attrition patterns, workforce risk hotspots, and retention insights at Palo Alto Networks using HR analytics and data visualization techniques.

The objective is to identify departments, job roles, demographic groups, and workforce factors that contribute to employee attrition. The project helps HR teams make data-driven decisions to improve employee retention and workforce planning.

## Problem Statement

Employee attrition can lead to:

- Loss of critical technical expertise
- Increased hiring and onboarding costs
- Disruption in long-term projects
- Reduced team morale and productivity

This project provides a comprehensive analysis of workforce attrition patterns to identify high-risk areas and support proactive retention strategies.

## Project Objectives

- Calculate overall attrition rate

- Evaluate overtime and travel impact
- Discover workforce risk hotspots
